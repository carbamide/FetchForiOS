Overview

The bread and butter of the application. DetailViewController is a UIViewController subclass that conforms to UISplitViewControllerDelegate, UITextFieldDelegate, and UITextViewDelegate. DetailViewController is responsible for the direct manipulation of the currently selected Url object as well as the creation of new Url objects.

DetailViewController is also responsible for the actual fetch action, as well as handling the data that occurs from that fetch action.
